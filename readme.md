## VR Museum

This is a project for Udacity Nanodegree program, made by Samantha Zhang, voiceovered by Yisi Lu.

Video Walkthrough: [https://youtu.be/8wpL7rX-_S0](https://youtu.be/8wpL7rX-_S0)

![](Screenshot_2.png)

### Platform

* Unity 2017.1.0p4

* Google VR SDK v1.60.0
